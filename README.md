<h3>课程介绍：</h3>
<p>LIBEVENT：是一款事件驱动的网络开发包，由于采用C语言开发体积小巧，跨平台，速度极快．课程中讲解分析 LIBEVENT原理，跨平台编译事件1O、缓冲1O处理。讲解HTTP服务端开发示例，HTTP客户端请求开发示例，最后基于 LIBEVENTT创建线程池C＋＋框架，并用此框架完成FTP服务器的登录、目录访问、文件上传下载能功能。</p>
